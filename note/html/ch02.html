2章のメモ

集合とは？

  • 要素の重複が無い
  • 順序が無い

定義 2.1.1

リストを集合とみなすと、だいたいこんな感じ。

明示的に要素を列挙

[1], [1,2], [1,2,3]

集合ではない例

[1,1] -- 要素が重複しているため

内包表記

[ ... | x <- xs ]

空集合

[]

集合 S と集合 T の差集合

S \\ T

具体例

>>> impport Data.List
>>> [1,2,3] \\ [1]
[2,3]

>>> [1,2,3] \\ [4,5]
[1,2,3]

>>> [1,2,3] \\ [1,2,3,4]
[]

集合の大きさ

length s

具体例

>>> length [1,2,3]
3

>>> length []
0

集合 S の冪集合

  • Haskell 順列・組み合わせ・べき集合・直積 から拝借。 
powerset :: [a] -> [[a]]
powerset [] = [[]]
powerset (x:xs) = [x:ps | ps <- powerset xs] ++ powerset xs

具体例

>>> powerset []
[[]]

>>> powerset [1]
[[1],[]]

>>> powerset [1,2]
[[1,2],[1],[2],[]]

>>> powerset [1,2,3]
[[1,2,3],[1,2],[1,3],[1],[2,3],[2],[3],[]]

定義 2.1.2

自然数の集合

n = [0, 1, 2, 3, 4, 5, ...]

可算とは？

  • 集合の要素が自然数と一対一に対応付けられるとき、その集合は可算であると言う。
